Project Name 		:	Beat the Trains

Category 		:	Android Application

Who are the users ?
	Anybody who has an android phone and wants to check on the local train Details 
can make use of 'Beat the Trains' app to get details such as nearest station and the 
timings of the next train.

Project Secification	:
	1) Displays timetables for all the stations.
	2) Locates nearest train station. 
	3) Provides approximate time left for next trains arrival.
	4) Enables user to set remainders for train arrivals.
